Privacy Policy for CoverCraft Chrome Extension

This Privacy Policy outlines the data collection and usage practices for the CoverCraft Chrome Extension ("Extension"). The Extension is designed to generate cover letters based on job board job descriptions, and it collects certain data to enhance its functionality.

Information We Collect

  1. User About Data

    We collect and store the user's professional experience and background ("User About"). This information is used to tailor and generate cover letters based on the user's skills, qualifications, and experiences.

  2. Active Tab Data

    We collect data related to the currently active tab in the user's browser. This data includes job description information from supported job board websites. The collected data is used to create customized cover letters for the job listings the user is interested in.

Usage of Collected Information

The data collected is used exclusively within the Extension and is never shared or transmitted outside of the user's local environment.

  • User About Data: The User About data is utilized to create cover letters that are specific to the user's professional background, skills, and experiences.
  • Active Tab Data: The active tab data is used to extract job descriptions from supported job board websites. This information is crucial in generating accurate and tailored cover letters for the user.

Data Storage

All user data, including User About information, is stored locally within the user's browser using Chrome Storage API. We do not have access to this data, and it is not transmitted to any external servers.

Security

We prioritize the security of user data and have implemented appropriate technical and organizational measures to safeguard the data collected by the Extension.

Consent

By installing and using the CoverCraft Chrome Extension, you consent to the data collection and usage practices described in this Privacy Policy.

Changes to this Privacy Policy

We may update this Privacy Policy as needed to reflect changes in the Extension's functionality or to comply with legal requirements. Users are encouraged to review this Privacy Policy periodically for any updates.
